# FlaskFileServer
사용자 관리와 프로그램을 배포하기 위한 간단한 서버

# 설치

윈도우  
```
#가상환경 생성 후 진입
python -m venv venv
venv\Scripts\activate

#종속성 설치
python -m pip install -r requirements.txt

```

